## Why Use This

WooCommerce's WordPress admin is powerful, but getting clean analytics out of it has always required extra plugins, CSV exports, or digging through multiple dashboard screens. This MCP server cuts through all of that.

**What you get:**

- **Instant WooCommerce analytics** — ask about revenue, orders, product performance, and customers without logging into WordPress.
- **No setup complexity** — no WooCommerce REST API keys or plugin configuration required; connect through Two Minute Reports in minutes.
- **Multi-store support** — analyze multiple WooCommerce stores from a single conversation.
- **Cross-platform context** — combine with 22+ other sources like Google Analytics 4, Klaviyo, and Facebook Ads to connect store data with traffic and email performance.

## What You Can Ask

### Sales & Revenue

```
What was my total revenue last month and how does it compare to the same period last year?
```

```
Show me daily revenue for the last 30 days.
```

### Orders

```
How many orders did I receive last week vs the week before?
```

```
What is my average order value this quarter compared to last quarter?
```

### Product Performance

```
Which 10 products drove the most revenue in the last 90 days?
```

```
What are my top and bottom performing products by units sold this month?
```

### Customer Insights

```
How many new vs returning customers made purchases last month?
```

```
What percentage of my revenue comes from repeat customers this quarter?
```

### Refunds & Returns

```
What was my refund rate last month?
```

```
Which products had the most refund requests this quarter?
```

### Coupon & Discount Analysis

```
How much revenue was discounted last month and what is my discount rate?
```

```
Which coupon codes were used the most this quarter?
```

### Inventory

```
Which products have fewer than 10 units left in stock?
```

```
Show me products that went out of stock in the last 30 days.
```
